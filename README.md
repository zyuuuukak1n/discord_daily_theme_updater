Discord Server Asset Automator
このスクリプトは、特定の日付に基づいた画像ファイルを使用して、Discordサーバーのアイコンとバナーを自動的に更新するためのPythonツールです。

概要
実行時の日付（YYYYMMDD形式）に一致する画像ファイルをディレクトリ内から自動で読み込み、指定したDiscordサーバーへ適用します。毎日、あるいは特定の日付にサーバーの外観を自動で変更したい場合に最適です。

主な機能
自動ファイル参照: 実行日の日付に基づいたファイル名（YYYYMMDD.gif および banner_YYYYMMDD.gif）を自動的に検索します。

一括更新: アイコンとバナーを一度のAPIコールで同時に更新します。

エラーハンドリング: 権限不足、サーバーブーストレベル不足（バナー変更に必要）、ファイル不在などのエラーを詳細にログ出力します。

セキュアな設計: トークンなどの機密情報は.envファイルで管理します。

セットアップ
1. 依存ライブラリのインストール
リポジトリをクローンした後、以下のコマンドで必要なライブラリをインストールしてください。

Bash
pip install -r requirements.txt
2. 環境変数の設定
.env.exampleをコピーして.envファイルを作成し、各自の情報を入力してください。

Bash
cp .env.example .env
DISCORD_BOT_TOKEN: Discord Developer Portalで発行したBotのトークン。

DISCORD_SERVER_ID: 変更を適用したいサーバーのID。

3. 画像の準備
スクリプトと同じディレクトリに、実行する日付に合わせた画像ファイルを配置します。

アイコン: 20260317.gif（例：2026年3月17日に実行する場合）

バナー: banner_20260317.gif

※スクリプトは.gif形式を想定していますが、ファイル名が一致していれば他の形式（.png, .jpgなど）でもコードを微調整することで対応可能です。

使い方
以下のコマンドを実行します。

Bash
python change_icon.py
注意事項
権限: Botには対象サーバーでの「サーバー管理」権限が必要です。

サーバーブースト: バナーの変更を適用するには、サーバーがブーストレベル2以上である必要があります。

レート制限: Discord APIのレート制限に注意して実行してください（頻繁な変更は推奨されません）。
